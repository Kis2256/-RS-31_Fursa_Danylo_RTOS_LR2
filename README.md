ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК

Лабораторна робота №2:  Програмування портів вводу-виводу загального призначення на основі прототипу операційної системи нескінченного циклу опитування 
реального часу.

Обладнання: STM32H743VIT6, CubeMX, STM32CubeIDE.

Мета: Ознайомитися з принципами програмування GPIO STM32F411 у режимі реального часу 
на основі нескінченного циклу опитування.

Завдання до лабораторної роботи:

Варіант 7. Запрограмувати світлодіоди PD12 та PD13 з частотою повторення імпульсів 0.5 Гц, 
тривалість імпульсу 200 мс. Зчитати стан кнопки PA0 та відобразити його на світлодіоді PD13

Загальний опис функціоналу 💡

- Тактування: Використовується внутрішній RC-генератор (HSI). Системний таймер (SysTick) налаштовано на генерацію переривань кожну 1 мс для забезпечення роботи функції HAL_GetTick().
- Входи: Вивід PC13 налаштовано як дискретний вхід із програмною підтяжкою до землі (Pull-Down) для постійного опитування стану кнопки.
- Виходи: Виводи PD12 та PD13 налаштовано як двотактні виходи (Push-Pull) для керування індикацією (світлодіодами).
- Таймінг: Робота базується на порівнянні поточного системного часу із заданими константами. Програма формує робочий цикл із частотою 0.5 Гц (загальний період 2000 мс)
та тривалістю активного імпульсу 200 мс без зупинки процесора.
- Формування імпульсів: Генерація логічної '1' протягом 200 мс та логічного '0' протягом наступних 1800 мс на виводах керування.
- Обробка дискретних сигналів: Безперервне зчитування логічного рівня порту PC13. Відповідно до алгоритму здійснюється паралельне керування станом виводу PD13 на основі натискання кнопки та таймера.

Схема підключення ⚙️
   
- LED1 (PD12): GND → LED → резистор (220 Ом) → PD12
- LED2 (PD13): GND → LED → резистор (220 Ом) → PD13
- Button (PC13): VDD (+3.3V) → кнопка → PC13 (з внутрішньою підтяжкою Pull-Down)
